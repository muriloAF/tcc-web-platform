# Capstone Project – Web Platform for Women in the Informal Labor Market

This repository contains my capstone project developed during the Technical Course in Systems Development at ETEC Itaquaquecetuba. The project aims to support women working in the informal labor market through a digital solution, applying web development and database integration concepts. 

## Project Objective
The objective of this project is to provide a digital solution that helps women in informal jobs gain visibility, access opportunities, and manage their services more efficiently.

## Features
-User can register as client or service providers;
-Service register with photos and descriptions;
-Service require system;
-Evaluation system;
-Chat;
-Courses

## Technologies Used
-HTML5;
-CSS3;
-PHP;
-JavaScript;
-MySQL;
-Ajax

## Project Structure
/css -> stylesheet;
/html -> pages with views;
/img -> images from the site;
/ImgBannersPrestadoras -> banner images from service providers;
/ImgPerfilCliente -> profile picture from clients;
/ImgPerfilPrestadoras -> profile picture from service providers;
/js -> back-end files in JavaScript;
/php -> connections with database and back-end systems in php;
/phpMailer -> phpMailer library, it's used to send e-mails;
/sounds -> sound packs made by Lucas Jacob dos Santos;
/sql -> database files;
/uploads -> File upload chat

## How to Run the Project
### Website
  -https://avenasing.rf.gd/html/Pagina_Inicial.html
### At your local storage
  -1. Clone this repository
  -2. Configure the database
  -3. Run the application using a local server

# Projeto de Conclusão de Curso – Plataforma Web para Mulheres no Trabalho Informal

Este repositório contém meu projeto de conclusão de curso desenvolvido durante o Curso Técnico em Desenvolvimento de Sistemas na ETEC Itaquaquecetuba. O projeto tem como objetivo apoiar mulheres que atuam no mercado de trabalho informal por meio de uma solução digital, aplicando conceitos de desenvolvimento web e integração com banco de dados.

## Objetivo do Projeto
O objetivo deste projeto é fornecer uma solução digital que ajude mulheres em trabalhos informais a obter maior visibilidade, acessar oportunidades e gerenciar seus serviços de forma mais eficiente.

## Funcionalidades
- Cadastro de usuários como clientes ou prestadoras de serviço;
- Cadastro de serviços com fotos e descrições;
- Sistema de solicitação de serviços;
- Sistema de avaliação;
- Chat interno;
- Cursos.

## Tecnologias Utilizadas
- HTML5;
- CSS3;
- PHP;
- JavaScript;
- MySQL;
- Ajax.

## Estrutura do Projeto
/css → folhas de estilo;  
/html → páginas e views do sistema;  
/img → imagens do site;  
/ImgBannersPrestadoras → banners das prestadoras de serviço;  
/ImgPerfilCliente → imagens de perfil dos clientes;  
/ImgPerfilPrestadoras → imagens de perfil das prestadoras;  
/js → arquivos JavaScript;  
/php → conexões com banco de dados e sistemas back-end em PHP;  
/phpMailer → biblioteca PHP Mailer utilizada para envio de e-mails;  
/sounds → pacotes de sons criados por Lucas Jacob dos Santos;  
/sql → arquivos de banco de dados;  
/uploads → arquivos enviados no chat.

## Como Executar o Projeto
### Website
- https://avenasing.rf.gd/html/Pagina_Inicial.html

### Localmente
1. Clonar este repositório;
2. Configurar o banco de dados;
3. Executar a aplicação utilizando um servidor local.
